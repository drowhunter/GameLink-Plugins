    To enable telemetry in dirt 2
  1. open C:\Users\username\Documents\My Games\DiRT2\hardwaresettings\hardware_settings_config.xml
  2. find the line <udp enabled="true" extradata="1" ip="127.0.0.1" port="20777" delay="1" >
  3. set udp enabled to true.

Working with External Programs

If you are using other software that uses telemetry like Simhub, you can have that software forward the telemetry on a different port. In which case you may enter the alternative port in the plugin configuration tab
