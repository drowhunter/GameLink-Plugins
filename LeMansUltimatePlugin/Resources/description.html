Le Mans Ultimate Telemetry Mod Installation Instructions

Install the Le Mans Ultimate GameLink Plugin

Copy the following files into the Yaw GameLink\Plugins folder:

  1. rF2SharedMemoryNet.dll
  2. LMUPlugin.dll
  3. Microsoft.Extensions.Logging.Abstractions.dll

Install the rFactor2SharedMemoryMapPlugin64 Telemetry Plugin

  1. Copy rFactor2SharedMemoryMapPlugin64.dll (included) to Le Mans Ultimate\Plugins directory.
  2. In game go to settings/Gameplay and enable the Telemetry
