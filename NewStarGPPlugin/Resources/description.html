Download Telemetry Mod here

Normal Install

  1. Install BepinEx v5 into your game folder. (click here)
  2. Place plugin dll file into BepinEx/plugins folder

UUVR install

  1. Download RaiPal. (click here)
  2. Scan for your games and then click on New Star GP.
  3. Install UUVR Mono Modern (which includes BepinEx)
  4. Install Telemetry Mod
    1. Click the 3 dots next to UUVR Mono Modern
    2. Click Open Mod Folder button

      The folder path should be something like: %APPDATA%\raicuparta\rai-pal\data\installed-mods\[ModId]\
    3. Place Telemetry Mod dll into BepinEx/plugins folder.
