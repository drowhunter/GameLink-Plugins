Easy Install

  1. Follow Steps 1-3 for UUVR insallation
    2. Then Click the Patch Game button and Enjoy

Normal Install (no VR)

  1. Download Telemetry Mod here
  2. Install BepinEx v5 into your game folder. (click here)
  3. Place plugin dll file from step 1 into [GameDir]/release/BepinEx/plugins folder

UUVR install

  1. Download RaiPal. (click here)
  2. Scan for your games and then click on New Star GP.
  3. Install UUVR Mono Modern (which includes BepinEx)
  4. Install Telemetry Mod
    1. Click the 3 dots next to UUVR Mono Modern
    2. Click Open Mod Folder button

      The folder path should be something like: %APPDATA%\raicuparta\rai-pal\data\installed-mods\[ModId]\
  5. Place Telemetry Mod dll into [UUVR_Mod_Folder]/BepinEx/plugins folder.
Note: The default profile was tuned with Yaw set to 15 degrees limit on pitch and roll
Press F5 in game for mod options
