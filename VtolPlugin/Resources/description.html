VTOL VR Mod Loader Update

To use this mod, subscribe to both the VTOL API and YAW Device VTOL VR Telemetry Mod on Steam under the VTOL VR Mod Loader's Workshop.

Setup Instructions

  • Start the VTOL VR Mod Loader to view all subscribed mods.
  • Enable "Load on Start" for both the YAW Telemetry Mod and the VTOL API Mod, or enable them manually in the game menu under "Mods" after starting the game.

Important

The VTOL API Mod must be enabled for the YAW Telemetry Mod to function correctly.
