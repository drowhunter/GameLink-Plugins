Star Wars X-Wing Alliance Upgrade

Install

  1. Install Star Wars X-Wing Alliance or Tie Fighter Total Conversion from Steam or elsewhere
  2. Download X-Wing Alliance Upgrade mod here, extract and install.
  3. Or download TFTC download and install it
  4. Once installed go into your modded game folder and edit CockpitLook.cfg
Make sure the following lines are present and configured as below: 
yawvr_enable = 0

UDP_telemetry_enabled = 1
UDP_telemetry_port = 20777
UDP_telemetry_server = 127.0.0.1
UDP_telemetry_format = JSON
UDP_telemetry_sparse = 0
