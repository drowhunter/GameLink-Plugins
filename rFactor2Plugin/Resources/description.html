Rfactor 2 Telemetry Mod Installation Instructions

Install the RFactor GameLink Plugin

Copy the following files into the Yaw GameLink\Plugins folder:

  1. rF2SharedMemoryNet.dll
  2. RFactor2Plugin.dll
  3. Microsoft.Extensions.Logging.Abstractions.dll

Install the RFactor Telemetry Plugin

  1. Copy rFactor2SharedMemoryMapPlugin64.dll (included) to Rfactor2\Plugins directory.
  2. In game go to settings/Gameplay and enable the rFactor2SharedMemoryMapPlugin64 plugin
